## Install / Instalación
#### OS X, Linux y Windows
*Abra un terminal y ejecute:*
```Shell
git clone http://github.com/pulidovpe/login-node.git
```
```Shell
cd login-node
```
```Shell
npm install
```
```Shell
node server.js
```
