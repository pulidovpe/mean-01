## Instalación
#### OS X, Linux y Windows
*Abra un terminal y ejecute:*
```ShellSession
git clone http://github.com/pulidovpe/login-node.git
```
```ShellSession
cd login-node
```
```ShellSession
npm install
```
```ShellSession
node server.js
```
